# Business Model: Technical testing and analysis

## Classification

- Repository: `cloud-itonami-isic-7120`
- ISIC Rev.5: `7120`
- Activity: technical testing and analysis -- physical, chemical and other testing of materials/products, and certification/inspection services
- Social impact: professional standards, data sovereignty, transparent audit

## Customer

- independent testing laboratories
- cooperative inspection bodies
- community quality-assurance programs

## Offer

- sample intake
- test-protocol proposal
- test-result/certification proposal
- immutable audit ledger

## Revenue

- self-host setup: one-time implementation fee
- managed hosting: monthly subscription per lab
- support: monthly retainer with SLA
- migration: import from an incumbent lab-information system
- per-test/certification fee

## Trust Controls

- no test result or certification is issued without human sign-off
- a fabricated test-protocol citation forces a hold, not an override
- every certification path is auditable
- emergency manual override paths remain outside LLM control
